# Prediction Market Knowledge Graph

Compiled overview for graph `eth.market.v1`. Entity pages are generated by the LLM as the wiki grows.

## Platforms

- **Polymarket** — crypto-focused prediction markets
- **Kalshi** — regulated US event contracts

## Decision timeline

See `log.md` for chronological ingests. Each entry maps to `data/decisions/dec-trd_*.json` and a `:Decision` node in Neo4j.

## Sample Cypher

```cypher
// All decisions, newest first
MATCH (d:Decision) RETURN d.decision_id, d.updated_at ORDER BY d.updated_at DESC

// Trades with thesis (BUY_YES / SELL_YES / etc.)
MATCH (m:Market)-[r]->(t:Trade)
WHERE r.thesis IS NOT NULL
RETURN m.node_id, type(r), r.thesis, r.conviction_level
ORDER BY r.timestamp DESC
LIMIT 10

// Correlated markets
MATCH (m:Market)-[r:CORRELATED_MARKET]-(other:Market)
RETURN m.node_id, other.node_id LIMIT 25
```
